# Portfolio Starter (TH)
วิธีใช้แบบรวดเร็ว:
1) เปิด `index.html` ได้ทันทีในเบราว์เซอร์
2) แก้ไขข้อมูลส่วนตัวในส่วน hero, about, skills, projects, experience และ action ของฟอร์มติดต่อ (Formspree)
3) อัพโหลดขึ้นโฮสติ้งสาธารณะ เช่น GitHub Pages, Netlify, Vercel

คุณสมบัติ:
- Responsive, รองรับ light/dark
- เมนูมือถือ, smooth scroll
- SEO meta + OG image
- โครงสร้างพร้อมแก้ไขได้ง่าย ไม่พึ่ง framework
